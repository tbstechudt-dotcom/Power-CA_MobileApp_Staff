│ AUTOMATED SCHEDULE                                                           │
└──────────────────────────────────────────────────────────────────────────────┘

  10:00 AM  →  Forward Full Sync        (Desktop → Supabase, all records)
  12:00 PM  →  Forward Incremental      (Desktop → Supabase, changed only)
  5:00 PM   →  Forward Incremental      (Desktop → Supabase, changed only)
  5:30 PM   →  Reverse Sync             (Supabase → Desktop, mobile data)

  Note: System must be ON and running at these times!

┌──────────────────────────────────────────────────────────────────────────────┐
│ ONE-TIME SETUP (5 minutes)                                                   │
└──────────────────────────────────────────────────────────────────────────────┘

  1. Open PowerShell as Administrator
     (Right-click PowerShell → "Run as Administrator")

  2. Run this command:
     cd "D:\PowerCA Mobile"
     powershell -ExecutionPolicy Bypass -File batch-scripts\automated\setup-windows-scheduler.ps1

  3. Verify tasks created:
     schtasks /Query /TN PowerCA*

  4. You're done! Syncs will run automatically.

┌──────────────────────────────────────────────────────────────────────────────┐
│ TESTING BEFORE AUTOMATION                                                    │
└──────────────────────────────────────────────────────────────────────────────┘

  Use the interactive menu for manual testing:

  Double-click: batch-scripts\manual\sync-menu.bat

  This gives you a menu to choose which sync to run manually.

┌──────────────────────────────────────────────────────────────────────────────┐
│ MONITORING                                                                   │
└──────────────────────────────────────────────────────────────────────────────┘

  View logs:
    dir /b /o-d logs\*.log                    (list recent logs)
    type logs\forward-sync-full_*.log         (view specific log)
    findstr /I "ERROR" logs\*.log             (search for errors)

  Check scheduled tasks:
    taskschd.msc                              (open Task Scheduler GUI)
    schtasks /Query /TN PowerCA*              (command line)

  Test a task manually:
    schtasks /Run /TN "PowerCA_ReverseSync_Daily"

┌──────────────────────────────────────────────────────────────────────────────┐
│ USEFUL COMMANDS                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

  # Disable a task
  schtasks /Change /TN "PowerCA_ForwardSync_Full" /DISABLE

  # Enable a task
  schtasks /Change /TN "PowerCA_ForwardSync_Full" /ENABLE

  # Delete all tasks
  schtasks /Delete /TN "PowerCA_ForwardSync_Full" /F
  schtasks /Delete /TN "PowerCA_ForwardSync_Incremental" /F
  schtasks /Delete /TN "PowerCA_ReverseSync_Daily" /F
